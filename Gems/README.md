## Gemini Context Files

This folder contains a collection of curated context files designed to enhance and customize your experience with Gemini. 

### How to Use
1.  **Select a file** from this repository that matches the persona or utility you need.
2.  **Copy the content** of the `.md` or `.txt` file.
3.  **Paste it** into the "System Instructions" or "Custom Instructions" field when creating a new Gem in the Gemini interface.
